✨ 主な特徴

ビジュアルプログラミング

直感的なドラッグ＆ドロップ操作で開発可能。複雑な構文を覚える必要はなく、ロジックの構築に集中できます。

豊富な機能ブロック

  • メッセージ受信時の応答
  • スラッシュコマンド (/command)
  • 埋め込みメッセージ (Embed)
  • プレフィックスコマンド (!ping)

Python コード生成

業界標準の discord.py ライブラリに対応したコードを自動生成。生成されたコードはそのまま実行可能です。

モダンな UI

Tailwind CSS による洗練されたデザインと、目に優しいダークモード対応で、快適な開発体験を提供します。
